# Project Overview

This is a Next.js project bootstrapped with `create-next-app`. It uses TypeScript, Tailwind CSS, and includes several UI components from Radix UI. The project is set up with the default Next.js 14 structure, including the App Router.

## Building and Running

To get the development server running, use one of the following commands:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Other Commands

*   **`npm run build`**: Builds the application for production.
*   **`npm run start`**: Starts a production server.
*   **`npm run lint`**: Lints the codebase using ESLint.

## Development Conventions

*   **Framework**: [Next.js](https://nextjs.org/)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **UI Components**: [Radix UI](https://www.radix-ui.com/)
*   **Linting**: [ESLint](https://eslint.org/)

The project follows the standard Next.js project structure. The main application code is located in the `src` directory. The `app` directory contains the pages and layouts for the App Router. Reusable UI components are located in the `src/components/ui` directory.
